實現 UITableView 下拉更新

雖然 UITableView 本身並沒有提供任何 refresh 的方法，但可以從他的super class - UIScrollView 裡取用。
我們要使用 UIScrollView 的原生 property - refreshControl，來指定一個 UIRefreshControl 給 table view。

在實作前可以先看看官方文件，裡面有很清楚的講解。

refreshControl
UIRefreshControl

首先建立一個 table view ，並給他一些初始資料。

接下來在 viewDidLoad 裡指定一個 UIRefreshControl 給 table view 的 refreshControl，並添加一個動作。
當 refreshControl 的值不為 nil 時，下拉 table view 就會有 refresh control 出現，並在下拉到一定的 offset 時執行指定的動作(如果有指定的話)。

在這裡我們隨機修改資料筆數，重新生成 data ，重整 table view 並隱藏 refreshControl。

如此一來就可以用下拉達到更新 table view 的效果。
